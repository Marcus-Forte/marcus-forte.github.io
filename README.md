# marcus-forte.github.io

Personal site built with [Astro](https://astro.build).

## Build & Deploy

Install dependencies:

```bash
npm install
```

Build the static site:

```bash
npm run build
```

Output is in `dist/`. GitHub Pages should be configured to serve from that directory (or use the GitHub Actions workflow to deploy automatically).

## Local Development

```bash
npm run dev
```
