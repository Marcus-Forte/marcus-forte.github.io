Portable 3D Scanner

Developed a portable 3D scanner for capturing real-world objects and creating 3D models in real time. The scanner uses a custom-made, decoupled SLAM algorithm that turns a sensor fusion method into an elegant non-linear optimization problem.The scanner device is as minimal as possible, with a single 3D LiDAR sensor with integrated IMU, a single-board computer, and a battery.
