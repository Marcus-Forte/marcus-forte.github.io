Delta Embedded Linux Biosensing System

Senior Software Designer

Sioux Technologies

04/2024 - 08/2024

Description

Designed and developed an embedded Linux C++ application for photonics-based, label-less biosensing equipment. The system controls infrared laser pulses and peripherals including temperature, pressure, and cylinder valves, while processing high-speed infrared-camera images. Built on a Yocto meta-project for the target image and supplied Python test scripts for test and integration engineers.
